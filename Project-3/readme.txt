1. use command below , to generate parser and lexer
	java -cp ./antlr-3.5.2-complete.jar org.antlr.Tool myChecker.g

2. then , use command below , to compile
	javac -cp ./antlr-3.5.2-complete.jar *.java

3. you can also use "make" to accomplish step1 and step2 , and use "make clean" to remove files that generated by makefile

4. use command below , to execute with .c file
	java -cp ./antlr-3.5.2-complete.jar:. myChecker_test test1.c
	java -cp ./antlr-3.5.2-complete.jar:. myChecker_test test2.c
	java -cp ./antlr-3.5.2-complete.jar:. myChecker_test test3.c

---------------------------------------------------------------------------

execute with test1.c and test2.c will have no output , because TRACEON = false
and there is no error

test3.c is an example for generate error message (as you provided project3_PPT_v1.pdf)
and there are comments in test3.c
