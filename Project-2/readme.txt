1. use command below , to generate tokens and java code
	java -cp ./antlr-3.5.2-complete.jar org.antlr.Tool myC_hw2.g

2. then , use command below , to compile
	javac -cp ./antlr-3.5.2-complete.jar *.java

3. you can also use "make" to accomplish step1 and step2 , and use "make clean" to remove files that generated by makefile

4. use command below , to execute with .c file
	java -cp ./antlr-3.5.2-complete.jar:. myC_hw2_test test1.c
	java -cp ./antlr-3.5.2-complete.jar:. myC_hw2_test test2.c
	java -cp ./antlr-3.5.2-complete.jar:. myC_hw2_test test3.c
